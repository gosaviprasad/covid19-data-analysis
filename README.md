covid19-data-analysis
A Python-based dashboard for visualizing COVID-19 trends using Pandas and Matplotlib.

## 🚀 Features
* **Data Cleaning:** Automated processing of raw COVID-19 datasets.
* **Trend Analysis:** Visualization of infection, recovery, and mortality rates.
* **Interactive Maps:** (Optional) Geospatial heatmaps of global cases.

## 🛠️ Tech Stack
* **Language:** Python 3.x
* **Libraries:** Pandas, Matplotlib, Seaborn, NumPy

## 📋 Prerequisites
Before running the project, ensure you have the following installed:
```bash
pip install pandas matplotlib seaborn
